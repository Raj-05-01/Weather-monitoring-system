# Realtime-weather-system
## Instructions to run
### Install dependencies
1. clone the repo.
2. Install the dependencies
 #### In Frontend
 ```
 npx create-react-app
 ```
 #### In backend
 ```
 npm init -y
 npm install axios cors dotenv epxress mongoose
 ```
   ### Running application
   1. open terminal
   2. Run Command
   ```
   cd backend
   Node server.js
   ```
   3. open another terminal 
   4. Run Command
   ```
   cd frontend
   npm start
   ```

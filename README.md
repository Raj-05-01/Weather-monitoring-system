# Weather-monitoring-system
## Description 
A real time weather Application to check weather data and wearher summary.
## Built with 
- Javascript
- Node.js
- React
- Mongodb
## Getting started
### Prerequisites
  - Make a directory backend
```
    cd backend
    Npm init -y
```
- Make a directory frontend
```
  cd frontend
  npx create-react-app
```
